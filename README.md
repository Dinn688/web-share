# 邻享（LinShare）V1.0

邻享是一款面向内部网络场景的文件分享系统，支持通过浏览器使用同一内网 IP 进行文件上传、下载、删除与批量操作。界面遵循苹果风格设计语言，默认全中文交互。

## 功能概览

- 无需登录：同一内部网络内设备输入访问地址即可使用。
- 上传能力：支持点击上传、拖拽上传，支持多文件与分片续传（暂停后继续）。
- 下载能力：支持单文件下载（显示进度、暂停、继续、取消）与批量压缩下载。
- 删除能力：支持单个删除与批量删除，删除前二次确认。
- 文件管理：支持按上传时间/文件大小/文件名称排序，支持关键词搜索。
- 操作日志：记录上传、下载、删除行为，便于追溯。

## 运行环境

- Node.js 18 及以上
- Windows 10+ / Ubuntu / CentOS 等主流 Linux 发行版

## 快速启动

### 1. 安装依赖

```bash
npm install
```

### 2. 启动服务

```bash
npm start
```

启动后控制台会显示：
- 本机访问地址（例如 `http://127.0.0.1:5832`）
- 内网访问地址（例如 `http://192.168.1.20:5832`）

同一内部网络内的电脑、手机、平板通过浏览器输入内网地址即可访问。

## 配置项（可选）

- `PORT`：服务端口，默认 `5832`
- `HOST`：监听地址，默认 `0.0.0.0`
- `UPLOAD_CHUNK_LIMIT_MB`：单个上传分片最大值（MB），默认 `16`

示例：

```bash
PORT=6000 UPLOAD_CHUNK_LIMIT_MB=32 npm start
```

## 目录说明

- `server.js`：后端服务入口（上传、下载、删除、日志、内网访问限制）。
- `public/`：前端页面与交互逻辑。
- `data/files/`：已上传文件存储目录。
- `data/temp/`：上传分片临时目录。
- `data/db.json`：元数据（文件、上传会话、日志）。

## 部署说明

### Windows

1. 安装 Node.js（建议 LTS 版本）。
2. 在项目目录执行：
   - `npm install`
   - `npm start`
3. 放行防火墙端口（默认 5832），确保局域网设备可访问。

### Linux（Ubuntu / CentOS）

1. 安装 Node.js（建议使用 NodeSource 或发行版稳定仓库）。
2. 在项目目录执行：
   - `npm install`
   - `npm start`
3. 放行防火墙端口（例如 `ufw allow 5832` 或对应发行版防火墙命令）。
4. 建议通过 `pm2` 或 `systemd` 做进程守护。

## 常见问题排查

- 无法访问页面：
  - 检查访问设备是否与服务器处于同一内部网络。
  - 检查访问地址和端口是否正确。
  - 检查防火墙是否放行服务端口。
- 上传中断：
  - 可在上传任务中点击“继续”进行续传，无需重传全部文件。
- 删除失败提示“文件正在被下载”：
  - 等待下载任务结束，或取消对应下载后再删除。

## 安全说明

- 默认仅允许内网地址访问（私有网段与本机回环地址）。
- 文件元数据保留 SHA-256 摘要，用于完整性校验与追溯。
- 删除操作强制二次确认。
